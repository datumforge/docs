This section encompasses the features / system functionality related to Digital Infrastructure management, which generically encompasses:

- Domain management
- Network asset management (IP blocks, virtual and physical devices)
- Infrastructure asset management (servers, storage, virtual and physical)
- Systems Assets (third party systems, in-house systems)
- Logical Assets
- Artifact management (systems and actual produced software artifacts)
